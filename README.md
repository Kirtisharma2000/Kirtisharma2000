Classifies messages as spam or ham with **97% accuracy**. Built an ensemble Voting Classifier (SVM + Decision Tree + Extra Trees), handled full text preprocessing pipeline with NLTK and TF-IDF, and deployed it as a live Streamlit web app.

A hybrid ML + rule-based system that reads your journal entry and metadata (sleep, stress, energy) to predict your emotional state and tell you what to do about it — and when. Includes uncertainty modeling and an ablation study comparing text-only vs text+metadata approaches.
